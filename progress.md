### What was implemented

**1. `Cargo.toml` changes**
- Bumped `oxichrome` from `"0.1"` → `"0.2"` 🎉 (now using v0.2.0 of oxichrome, oxichrome-core, oxichrome-macros, oxichrome-build)
- Added `leptos = { version = "0.8", features = ["csr"] }`

**2. `src/types.rs` — TabInfo enhanced**
- Added `group_id: i32` field with `#[serde(default = "no_group")]` — defaults to `-1` (Chrome's ungrouped tab value)
- Helper `fn no_group() -> i32 { -1 }` for serde default
- Fully backward-compatible: all 60 existing tests pass

**3. `src/popup.rs` — Full Leptos popup component** (`#[oxichrome::popup]`)
- **Header**: Brand logo (3 coloured bars), title "Tab Cleanner", dynamic subtitle showing "X groupes · Y onglets", gradient button "Ranger maintenant", last-run info text
- **Async data flow**:
  - On mount: sends `PopupCommand::GetState` via `runtime::send_message()`, deserialises `MessagingResponse` then parses JSON `GroupState`
  - Also queries current window tabs via `tabs::query()` with enhanced `TabInfo` (includes `group_id`)
  - Matches tabs to stored groups by `StoredGroup.group_id` ↔ `TabInfo.group_id`
  - Tabs with `group_id = -1` go to "Non rangés" section
- **Groups list**: Expandable accordion per group:
  - Chevron toggle, colour dot, display name (display_name or fallback to domain name), tab count badge
  - Expanded body: 8-colour palette (no grey, hex values matching reference), "Thème" field with "bientôt" badge, tab list with title + URL
- **RunGrouping**: Button sends `PopupCommand::RunGrouping`, then auto-refetches state; button disabled + spinning animation during operation
- **Loading state**: Spinner + "Chargement des groupes…"
- **Error state**: Red error text with detail message (no panic, no unwrap)
- **Styling**: Dark theme matching `design-popup-reference.jsx` — exact colours, spacing, typography, scrollbar, transitions
- **No unwrap()** anywhere in runtime paths — all results handled with `map_err`/`ok_or_else`

**4. `dist/chromium/popup.html`**
- Width 384px, follows Oxichrome pattern: loads wasm + calls `__oxichrome_mount_popup()`
- Auto-generated by `cargo oxichrome build` in v0.2, but included manually for immediate use

**5. `dist/chromium/manifest.json`**
- Added `"action": { "default_popup": "popup.html" }` (auto-generated by v0.2 build when `#[oxichrome::popup]` detected)
